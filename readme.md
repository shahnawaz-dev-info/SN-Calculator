# SN Calculator

A modern and responsive calculator built using **HTML, CSS, and JavaScript**. This project performs basic arithmetic operations with a clean and user-friendly interface.

## 🚀 Features

- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Percentage (%)
- Decimal calculations
- Clear (AC) button
- Delete (⌫) button
- Keyboard support
- Responsive design
- Modern UI with hover effects

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 📁 Project Structure

```
Calculator/
│── index.html
│── style.css
│── script.js
└── README.md
```

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` or open it in your browser.
