# SN Calculator

A modern and responsive calculator built using **HTML, CSS, and JavaScript**. This project performs basic arithmetic operations with a clean and user-friendly interface.
🧮 SN Calculator is a simple, modern, and user-friendly calculator application designed for quick and easy calculations. It supports basic operations, percentage, decimal calculations, keyboard input, and a responsive interface. 🚀 This project focuses on creating a smooth and convenient experience for everyday calculations.

## 🌍 Live Preview
Experience SN Calculator live:

👉 Open SN Calculator https://sn-calculatorr.netlify.app

## 🚀 Features

- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Percentage (%)
- Decimal calculations
- Clear (AC) button
- Delete (⌫) button
- Keyboard support
- Responsive design
- Modern UI with hover effects

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
